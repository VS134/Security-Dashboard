# Security Dashboard

A lightweight, browser-based security toolkit built with **vanilla HTML, CSS, and JavaScript**. This project focuses on core cybersecurity concepts including password strength analysis, cryptographic hashing, and Unix file permissions.

---

## Features

### 1. Password Strength Checker
- Evaluates password strength based on:
  - Length
  - Character variety (uppercase, lowercase, digits, symbols)
- Provides feedback on missing criteria
- Designed to encourage stronger password practices

---

### 2. SHA-256 Hash Generator
- Converts any input string into a **SHA-256 hash**
- Uses the Web Crypto API
- Demonstrates one-way hashing used in secure systems

---

### 3. Linux Permissions Calculator & Visualizer
- Interactive grid for:
  - Owner, Group, Public
  - Read, Write, Execute
- Dynamically calculates:
  - Numeric permissions (e.g. `755`)
  - Symbolic format (e.g. `-rwxr-xr-x`)
- Displays the corresponding `chmod` command

---

## Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- No frameworks or external libraries
